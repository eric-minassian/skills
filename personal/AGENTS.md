# Personal Preferences

## TypeScript

- Never us `any`, `!`, compiler directives or other escape hatches unless 100% necessary or specifically instructed.

## Commands

- Don't run dev server commands (e.g., `pnpm run dev`).
- Don't run build commands unless specifically told to.
- Focus on checking commands like `pnpm run typecheck`, `pnpm run lint`, etc.

## Package Managers

- Use bun if the project already uses it, otherwise use pnpm.
- Never use npm or yarn.

## Tech Stack Preferences

When uncertain, prefer: Tailwind, Typescript, Pnpm, React, Vite, Playwright, AWS CDK.

## Code Style

- Always strive for concise, simple solutions.
- If a problem can be solved in a simpler way, propose it.

## Workflow

- If asked to do too much at once, stop and state that clearly.
